# tidy unmarked parameter estimates

tidy unmarked parameter estimates

## Usage

``` r
tidy.unm(X)

tidy.unm
```

## Format

An object of class `function` of length 1.

## Arguments

- X:

  an unmarked fit object

## Value

A data frame of parameter estimates similar to what broom::tidy() does
for lm/glm objects
